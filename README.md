📌 Project Title: Free Certificate Platforms Directory

This project is a Django-based web application that provides a curated directory of online learning platforms offering free courses with certificates. Instead of listing individual courses, the platform helps users discover trusted websites where they can learn and earn certifications at no cost.

🚀 Features
🌐 List of platforms offering free certificate courses
📖 Detailed descriptions of each platform
🔗 Direct links to explore courses on respective platforms
🎯 User-friendly and responsive interface
⚙️ Admin panel to manage and update platform data

🛠️ Tech Stack
Frontend: HTML, CSS, Bootstrap
Backend: Django (Python)
Database: SQLite
🎯 Objective

To simplify the search for reliable e-learning platforms that provide free certificates, helping students and learners save time and access quality education easily.
