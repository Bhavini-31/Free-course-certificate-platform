About Us

Welcome to CourseHub 🎓 – a free certificate course platform designed to help learners discover high-quality free courses from multiple trusted platforms in one place.

Our mission is to simplify learning by collecting resources from platforms like freeCodeCamp, Saylor Academy, Cisco Networking Academy, HP LIFE, and more.

CourseHub does not create courses. Instead, it acts as a collection platform where users can easily explore and access free learning opportunities with certificates.

Built using Python (Django), this project demonstrates real-world web development skills with a clean and responsive design.
